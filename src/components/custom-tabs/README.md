# Custom Tabs

This project is a reusable tab component that allows users to switch between different content views.

## Features

- Displays a list of tabs.
- Shows the content of the selected tab.
- Calls a callback function when the selected tab changes.

## Code

- `tabs.jsx`: The main tab component.
- `tab-test.jsx`: An example of how to use the tab component.
- `tabs.css`: The styles for the component.
