# Accordion

This project is a simple accordion component that allows users to show and hide content. It can be configured to allow single or multiple selections.

## Features

- Single selection: Only one item can be open at a time.
- Multiple selection: Multiple items can be open at the same time.

## Code

- `index.jsx`: The main component file.
- `data.js`: The data for the accordion items.
- `styles.css`: The styles for the component.
