# Star Rating

This project is a star rating component that allows users to select a rating from a given number of stars.

## Features

- Displays a configurable number of stars.
- Allows users to select a rating by clicking on a star.
- Highlights the selected stars and the stars before it.
- Shows a hover effect when the user moves the mouse over the stars.

## Code

- `index.jsx`: The main component that renders the stars and handles the user interaction.
- `styles.css`: The styles for the component.
