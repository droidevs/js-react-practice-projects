# Load More Data

This project demonstrates how to fetch data from an API and display it in a list. When the user clicks the "Load More Products" button, it fetches more data and appends it to the existing list.

## Features

- Fetches a list of products from the dummyjson API.
- Displays the products in a grid.
- "Load More Products" button to fetch and display more products.
- Disables the button when all products have been loaded.
- Loading state.

## Code

- `index.jsx`: The main component that fetches the data, manages the state, and displays the products.
- `styles.css`: The styles for the component.
