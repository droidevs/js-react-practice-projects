# Image Slider

This project is a reusable image slider component that fetches images from a given URL and displays them in a slideshow.

## Features

- Fetches images from a URL.
- Displays images in a slideshow with previous/next buttons.
- Includes circle indicators to show the current slide.
- Loading and error states.

## Code

- `index.jsx`: The main component that fetches the images and handles the slideshow logic.
- `styles.css`: The styles for the component.
