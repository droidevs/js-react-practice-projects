# QR Code Generator

This project is a simple QR code generator. The user can enter any text, and the application will generate a QR code for it.

## Features

- Generates a QR code from user-provided text.
- Uses the `react-qr-code` library to generate the QR code.
- The "Generate" button is disabled if the input is empty.

## Code

- `index.jsx`: The main component that handles the user input and generates the QR code.
- `styles.css`: The styles for the component.
