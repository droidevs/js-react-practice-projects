# Custom Modal Popup

This project is a reusable modal component that can be customized with different content for the header, body, and footer.

## Features

- Customizable header, body, and footer.
- Close button to dismiss the modal.
- Can be opened and closed programmatically.

## Code

- `modal.jsx`: The main modal component.
- `modal-test.jsx`: An example of how to use the modal component.
- `modal.css`: The styles for the component.
