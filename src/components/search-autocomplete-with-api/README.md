# Search Autocomplete with API

This project is a search autocomplete component that fetches a list of users from an API and displays suggestions as the user types.

## Features

- Fetches a list of users from the dummyjson API.
- As the user types in the search box, it filters the list of users and displays a dropdown of suggestions.
- When the user clicks on a suggestion, it populates the search box with the selected user.
- Loading and error states.

## Code

- `index.jsx`: The main component that fetches the user data, handles the search input, and filters the suggestions.
- `suggestions.jsx`: The component that displays the list of suggestions.
