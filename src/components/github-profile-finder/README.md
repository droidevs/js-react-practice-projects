# Github Profile Finder

This project allows users to search for a Github user by their username and displays their profile information.

## Features

- Search for a Github user by username.
- Displays the user's avatar, name, join date, public repos, followers, and following count.
- Links to the user's Github profile.

## Code

- `index.jsx`: The main component that handles the search input and fetches the user data from the Github API.
- `user.jsx`: The component that displays the user's profile information.
- `styles.css`: The styles for the component.
