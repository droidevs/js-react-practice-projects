# Tic-Tac-Toe Game

This is a simple Tic-Tac-Toe game built with React.

## How to use

To use this component, import it into your React application and render it.

```jsx
import TicTacToe from './components/tic-tac-toe';

function App() {
  return (
    <div className="App">
      <TicTacToe />
    </div>
  );
}

export default App;
```

The component handles the game logic, including:

-   Displaying the game board.
-   Handling user clicks to make a move.
-   Determining the winner of the game.
-   Detecting a draw.
-   Allowing users to restart the game.
